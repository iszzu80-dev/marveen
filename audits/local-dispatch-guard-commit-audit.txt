    actually process them.

      referential false positives from scrollback quoting the same
      phrase, including this commit's own diff).

      context saturation, closing the gap for both the inter-agent
      message-router and the scheduled-task runner in one place.

      capture -> loop-guard -> fresh restart -> recovery-context brief),
      debounced 5 minutes per session so repeated polling ticks don't
      spawn concurrent recoveries.

      case built from tonight's real captured incident shape.
